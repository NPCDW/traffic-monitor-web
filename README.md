# traffic-monitor-web

流量监控前端

后端地址: [traffic-monitor](https://github.com/NPCDW/traffic-monitor)

## 开发

先启动后端，或找一个后端的地址，将 `.env.development` 文件中 `VITE_DEV_SERVER_BASE_URL` 改为后端地址，或新建 `.env.development.local` 文件并将 `VITE_DEV_SERVER_BASE_URL` 改为后端地址

```bash
npm install -g pnpm
pnpm install
pnpm run dev
```
